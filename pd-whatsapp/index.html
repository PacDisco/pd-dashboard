How it works

This page uses Google Identity Services to authorize your Google account, then writes the contact via the People API people.createContact endpoint. Nothing is stored on the server — the access token lives only in this browser tab.

  • Sign in with Google (top right)
  • Fill in at least a name, email, or phone
  • Click Save contact

First-time setup: see add-contact/oauth-setup.md for creating the Google Cloud OAuth Client ID.
